# HyperFlow Evaluator Session Script

## Facilitator intro

“I’m going to show you a short interactive demo. Please use it like you’re seeing it for the first time and think out loud while you explore.”

## Important facilitator rule

Do **not** explain:

- what the “right” interpretation is
- what each scenario is supposed to mean
- what the evaluator should conclude

Only answer technical blockers that prevent the session from continuing.

## Session prompts

### Prompt 1 — first impression

“Take a look at this screen. What do you think this demo is showing?”

### Prompt 2 — first action

“What would you click or read first?”

### Prompt 3 — scenario meaning

“What do these scenario choices mean to you?”

### Prompt 4 — proof understanding

“What do you think the runtime summary is trying to prove?”

### Prompt 5 — interaction understanding

“After using the controls, what do you think HyperFlow is good at?”

## Neutral follow-ups

Use only if needed:

- “What made you think that?”
- “Was anything unclear?”
- “What would you rename here?”
- “What felt most useful?”

## Closing question

“If you had to explain this demo to someone else in one sentence, what would you say?”
