# vanilla-starter

Guided product-demo surface for the current HyperFlow proof slice.

## What it demonstrates
- scenario-first runtime storytelling
- Rust viewport/culling/hit-test proof through the SDK contract
- summary cards for visible nodes, viewport work, and render timing
- a lightweight guided interaction flow

## Run

From the repo root:

```bash
pnpm build:wasm:poc
pnpm serve:poc
```

Then open:

```
http://localhost:4173/examples/vanilla-starter/index.html
```
